# Updating GitHub Pages

1. Unzip this package.
2. Go to your GitHub repo.
3. Click **Add file > Upload files**.
4. Drag the extracted files/folders into GitHub.
5. Commit message: `Clean up usable SiteNav build`
6. Wait 1-3 minutes.
7. Refresh the site with Ctrl+F5.

If folders do not overwrite correctly, upload these manually:

- assets/plant-access-clean.png
- data/locations.json
- data/routes.json
- index.html
- style.css
- app.js
