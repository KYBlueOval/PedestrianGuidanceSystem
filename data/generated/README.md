# Generated DTTK data

This directory is reserved for sanitized PGS package data generated by DTTK.
Generated JSON is excluded from Git until its contents are reviewed for
public-release suitability.

PGS v10.1 recognizes these optional approved-network files:

- `pedestrian_network.json`
- `destination_node_crosswalk.json`

The 3D route is enabled only when `pedestrian_network.json` declares
`review.route_ready: true` and both selected destinations exist in the
crosswalk. Draft route-editor exports must first be compiled and approved with
DTTK Commit 04D.
