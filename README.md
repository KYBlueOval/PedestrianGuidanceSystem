# Ford Energy PGS — v5 Fixed Alignment

## Upload

1. Extract ZIP.
2. GitHub repo → Add file → Upload files.
3. Drag all extracted files/folders.
4. Commit changes.
5. Refresh GitHub Pages with Ctrl+F5.
